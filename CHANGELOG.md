# Changelog

All notable documented changes to the Windows Troubleshooter Toolkit are recorded here.

## 10.1 — Current documented release

### Included

- Expanded Windows diagnostics and guarded repair workflows
- Microsoft 365, Outlook, Teams, OneDrive and activation troubleshooting
- Network, DNS, DHCP, VPN, proxy, firewall and sharing checks
- Windows Update, service, startup, storage and performance workflows
- Entra ID, MFA, Conditional Access, BitLocker, Group Policy and domain-trust checks
- Printing, profile migration, browser, Autopilot, SCCM/MECM, WMI and Hyper-V support tools
- Audit logging and diagnostic evidence generation
- Launch helper for elevation and technician-friendly startup

### Documentation

- Added clearer requirements and run instructions
- Added safety and known-limitations guidance
- Added links to related focused single-run tools
- Added repository status and version badges

> Historical release details before version 10.1 were not previously maintained in a formal changelog.
